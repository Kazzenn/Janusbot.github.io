# หน้าแรก

## เริ่มต้นใช้งานบอทหลายบาทแบบง่ายและเร็ว

* ทำการชวนบอทหลายบาทเข้าดิส [ลิงก์ชวนบอท](https://discord.com/oauth2/authorize?client_id=483104273761304577&permissions=11534344&scope=bot%20applications.commands)
* เข้าห้องเสียง
* `/help` - คำสั่งทั้งหมด
* `/play` - คำสั่งเล่นเพลง
* `/skip` - คำสั่งข้ามเพลง
* `/leave` - เลิกใช้งานบอท
